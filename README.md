# Executas

beatmania IIDX INFINITAS の起動用スクリプトです。

個人的に使う用に書いているため、一部設定が直書きになっているので使う場合はいい感じに修正して使ってください。

# Installation

1. リポジトリをダウンロードする
2. `src`フォルダの中身を`Documents`フォルダにコピーする
   - `Documents/WindowsPowerShell/(以下略)` というディレクトリ構造になれば OK
3. bat ファイルを実行する
